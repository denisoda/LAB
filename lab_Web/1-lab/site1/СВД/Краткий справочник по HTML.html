Краткий справочник по HTML
Подготовлен по материалам Сапенаур С., Куэрсиа В. Справочник Web-мастера: Пер. с англ. - К.: BHV, 1997.

HTML - (Hypertext Markup Language, язык разметки документа) - это язык компоновки документов и спецификации гиперссылок, используемый для кодирования документов в системе World Wide Web. Документ, который написан на языке HTML представляет собой простой текст, в который вставлены флаги разметки, или теги (markup tags), в соответствии с которыми программа просмотра (броузер) отображает содержимое дакумента: текст, изображения и данные других типов.

Синтаксис HTML-тегов
HTML-тег состоит из имени, за которым может следовать список атрибутов тега.Текст тега заключается в угловые скобки (< и >). Простейший вариант тега - имя, заключенное в угловые скобки, например <HEAD> или < i >. Для более сложных тегов характерно наличие атрибутов, которые могут иметь конкретные значения, определенные автором, для видоизменения функции тега.

Атрибуты тега следуют за именем и отделяются друг от друга одним или несколькими знаками табуляции, пробелами или символами возврата к началу строки. Порядок записи атрибутов значения не имеет. Значение атрибута, если таковое имеется, следует за знаком равенства, стоящим после имени атрибута. Если значение атрибута слово или число, то его можно просто указать после знака равенства, не выделяя дополнительно. Все остальные значения следует заключать в одинарные или двойные кавычки. Длина значения атрибута не должна превосходить 1024 символа.

Чаще всего HTML-теги состоят из начального и конечного компонентов, между которыми размещаются текст и другие теги. Имя конечного тега идентично имени начального, но перед именем конечного тега ставится косая черта (/). Конечные теги не могут иметь атрибуты.

Здесь дается описание распространенных тегов и атрибутов HTML, которые в настоящее время используются в Web-документах. Существует множество различных броузеров, которые поддерживают не совпадающие между собой наборы тегов. Два самых популярных броузера – Netscape Navigator и Microsoft Internet Explorer – поддерживают практически все нестандартные расширения HTML. Наиболее распространенная версия броузера Netscape Navigator – Netscape Navigator 3.0. Перед описанием тегов и атрибутов, специфичных для этого броузера, стоит обозначение N3 Текущая версия Internet Explorer фирмы Microsoft – тоже 3.0. Ей соответствует обозначение IE3.

Описание структуры документа

<HTML>

<HTML>...</HTML>
Начальный и конечный теги всего HTML-документа
Атрибуты
version = строка
Указывается версия HTML, которая была использована для создания данного документа.

<head>

<head>...</head>
начальный и конечный теги заголовка документа.

<title>

<title>...</title>
Начальный и конечный теги названия HTML-документа.

<body>

<body>...</body>
начальный и конечный теги тела документа.
Атрибуты
alink = цвет
Установка цвета активных гипертекстовых ссылок в документе.
Backgroud = url
Указывается URL фонового изображения.
Bgcolor = цвет
Установка цвета фона документа.
Bgproperties = значение
IE2 и выше. Если значение равно fixed, запрещается прокрутка фонового изображения вместе с содержимым документа.
Leftmargin = значения
IE2 и выше. Установка размера (в пикселях) левого поля документа.
link = цвет
Установка цвета “непосещенных” гипертекстовых ссылок в документе.
Text = цвет
Установка цвета обычного текста в документе.
Topmargin = значения
IE2 и выше. Установка размера (в пикселях) верхнего поля документа.
vlink = цвет
Установка цвета “посещенных” ссылок в документе.

<style>

<style>...</style>
Только IE3. Заключает в себе список элементов стиля, которые необходимо использовать для форматирования документа. Блок <style> идет перед тегом <body> и вне всех прочих элементов, кроме <html>...</html>.

<meta>

<meta>...</meta>
Предоставление дополнительной информации о документе.
Атрибуты
content = строка
Указывается строка для мета-информации. Этот атрибут является необходимым.
Для операций “вытягивания” на стороне клиента посредством content = "n;url = url" броузеру дается указание загрузить указанный url через n секунд. Если URL не задан, перезагружается исходный документ. Необходимо использовать с http-equiv = ”refresh” в <meta>.
http-equiv = строка
Указывается эквивалентное HTTP-имя для данной метаинформации и дается команда серверу включить данное имя и содержимое в HTTP - заголовок для данного документа для передачи его клиенту.
При задании значение refresh в документе создается “клиентское вытягивание”.
Name = строка
Указывается имя метаинформации.

  Управление разрывом строк

<p> <p>...</p>
начальный и конечный теги абзаца
Атрибуты
align = тип
Задается способ выравнивания текста в абзаце: по левому краю (left), центру(center), или правому краю (right).
<hr> <hr> разрыв текущего текстового потока. В месте разрыва будет вставлена горизонтальная линейка.
Атрибуты
align = тип
Задается способ выравнивания линейки: по левому краю (left), центру (center), или правому краю (right).
Noshade
Запрещается использование объемного затенения при отображении линейки.
Size = пиксели
Установка толщины линейки равной целому числу пикселей.
Width = значение
Установка ширины линейки либо равной целому числу пикселей, либо в процентах от ширины страницы.
<div>

<div>...</div>
В документе создается раздел.
Атрибуты
align = тип
Текст в разделе будет выровнен по левому краю (left), центру(center), или правому краю (right).

<br>

<br>
Разрыв текущего текстового потока и возобновление его с начала следующей строки.
Атрибуты
clear = поле
Задается “обтекание” объекта, расположение которого указано значением данного атрибута (left, right или all). При переносе на новую строку текст будет размещаться так, чтобы объект оставался видимым.

<nobr>

<nobr>...</nobr>
В заключенном в теги тексте разрывы не допускаются.

<wbr>

<wbr>...</wbr>
Начальный и конечный теги точки возможного разрыва слова в разделе <nobr>.

<spacer>

<spacer>
Только N3. Вставить в текст разделитель.
Атрибуты
type = тип
Указывается тип разделителя: vertical - между двумя строками текста помещается область пустого пространства указанного размера; horizontal - область пустого пространства указанного размера помещается между словами или символами; block - вставка прямоугольной области (подобно <img>).
Size = n
Указывается (в пикселях) ширина разделителя типа horizontal или высота разделителя типа vertical.
Width = n
Указывается (в пикселях) ширина разделителя типа block.
Height = n
Указывается (в пикселях) высота разделителя типа block.
Align = значение
Указывается способ выравнивания разделителя типа block относительно окружающего текста. Значения такие же, как в теге <img>.

Выбор шрифта

<font>

<font>...</font>
Установка размера, цвета или гарнитуры заключенного в теги текста.
Атрибуты
color = цвет
Установка цвета заключенного в теги текста.
Fase = список
Установка гарнитуры заключенного в теги текста (устанавливается первый из указанных в разделенном запятыми списке имен шрифтов).
Size = значение
Размер шрифта устанавливается равным заданному абсолютному значению (от 1 до 7) или относительно размера <basefont> с помощью значений +n или -n.

<basefont>

<basefont>...</basefont>
Указывается размер шрифта для последующего текста.
Атрибуты
Size = значение
Установка размера базового шрифта. Диапазон - от 1 до 7. (Требуется; по умолчанию принимается значение 3).

<i>

<i>...</i>
Заключенный в теги текст будет отображаться в курсивном начертании.

<b>

<b>...</b>
Заключенный в теги текст будет отображаться жирным шрифтом.

<tt>

<tt>...</tt>
Заключенный в теги текст будет отображаться машинописным шрифтом.

<hn>

<hn>...</hn>
Заключенный в теги текст представляет собой заголовок уровня n. Возможные значения n - от 1 до 6.
Атрибуты
align = тип
Указывается способ выравнивания заголовка: по левому краю (left), центру(center) или правому краю (right).

<marquee>

<marquee>...</marquee>
IE2 и выше. Создается бегущая строка текста.
Атрибуты
align = позиция
Задается позиция бегущей строки по вертикали относительно текста, в котором она расположена: вверху (top), посередине (middle) или внизу (bottom).
Behavior = стиль
Стиль бегущей строки определяется как alternate(меняет направление, дойдя до границы), scroll(побуквенно исчезает дойдя до границы и также появляется с другой стороны) мнгновенно начинает движение снова) или slide(дойдя до границы мнгновенно начинает движение снова).
Bgcolor = цвет
Установка цвета фона бегущей строки.
Direction = направление
Задается направление прокрутки текста: влево (left) или вправо (right).
Height = значение
Задается высота (в пикселях) зоны бегущей строки.
Hspace = значение
Задается размещение слева и справа от бегущей строки областей свободного пространства (в пикселях).
Loop = значение
Установка числа повторов анимации бегущей строки; значение может быть целым или infinite.
Scrollamount = значение
Задается размер (в пикселях) области смещения текста при каждой операции прокрутки.
Scrolldelay = значение
Указывается задержка (в миллисекундах) между последовательными смещениями текста бегущей строки.
Vspace = значение
Задается размещение над бегущей строкой и под ней областей свободного пространства заданной высоты (в пикселях).
Width = значение
Задается ширина зоны бегущей строки (в пикселях).

<sub>

<sub>...</sub>
Заключенный в теги текст будет смещен вниз (нижний индекс).

<sup>

<sup>...</sup>
Заключенный в теги текст будет смещен вверх (верхний индекс).

<s>

<s>...</s>
Заключенный в теги текст будет отображаться перечеркнутым горизонтальной линией.

<strike>

<strike>...</strike>
Заключенный в теги текст будет перечеркнут горизонтальной линией.

<em>

<em>...</em>
Для заключенного в теги текста в качестве дополнительного параметра форматирования задается курсивное начертание.

<strong>

<strong>...</strong>
Для заключенного в теги текста в качестве дополнительного параметра форматирования задается жирное начертание.

<big>

<big>...</big>
Заключенный в теги текст будет отображаться шрифтом большего размера.

<small>

<small>...</small>
Заключенный в теги текст будет отображаться шрифтом меньшего размера.

  Форматирование абзацев

<blockquote>

<blockquote>...</blockquote>
Заключенный в теги текст представляет собой блок цитаты.

<center>

<center>...</center>
Размещение заключенного в теги текста по центру.

<cite>

<cite>...</cite>
Заключенный в теги текст представляет собой цитату.

<code>

<code>...</code>
Заключенный в теги текст представляет собой фрагмент кода.

<comment>

<comment>...</comment>
IE2 и выше. Комментарий в тексте документа. Комментарии будут видимы в любом броузере. Для всех броузеров броузеров комментарии представляются в виде <!--текст комментария-->.

<dfn>

<dfn>...</dfn>
Заключенный в теги текст форматируется как определение.

<kbd>

<kbd>...</kbd>
Заключенный в теги текст выводится посимвольно (как при наборе с клавиатуры).

<multicol>

<multicol>...</multicol>
Только N3. Заключенные в теги код HTML и текст будут представлены в многостолбцовом формате. Текст и элементы будут распределяться в соответствии с указанным количеством столбцов так, чтобы элементы столбцов были приблизительно равной длинны.
Атрибуты
cols = n
Указывается количество столбцов. Этот атрибут является необходимым.
gutten = n
Указывается расстояние (в пикселях) между столбцами. По умолчанию - 10 пикселей.
width = n
Указывается ширина столбцов в пикселях.

<plaintext>

<plaintext>
Указывается, что остальную часть документа следует отображать без обработки, как предварительно отформатированный текст.

<pre>

<pre>...</pre>
Заключенный в теги текст будет отображаться так, как он был отформатирован предварительно, без обработки, с точным соблюдением переносов строк и интервалов.
Атрибуты
width = n
Броузер будет размещать текст так, чтобы в строке умещалось (если возможно) n символов.

<samp>

<samp>...</samp>
Заключенный в теги текст представляет собой шаблон

<span>

<span>...</span>
Только IE3. Заключенный в теги текст будет форматироваться с использованием таблицы стилей.
Атрибуты
style = элементы
Для текста в заданном интервале задаются элементы таблицы стилей.

<var> <var>...</var>
Заключенный в теги текст представляет собой имя переменной.

Списки

<ol> <ol>...</ol>
Начальный и конечный теги упорядоченного списка, содержащего пронумерованного (в порядке возрастания) элементы <li>.
Атрибуты
compact
Список будет представлен в более компактном виде.
Start = n
Список будет пронумерован начиная не со значения 1 (по умолчанию), а со значения n.
Type = формат
Установка формата нумерации данного списка: A (прописные буквы), a (строчные буквы), I (большие римские цифры), i (маленькие римские цифры), 1 (арабские цифры; по умолчанию).
<ul> <ul>...</ul>
Начальный и конечный теги неупорядоченного списка помеченных элементов <li>.
Атрибуты
compact
Список будет представлен в более компактном виде.
Type = формат
N2 и выше. Задается стиль меток для данного списка: circle (кружок), disk(диск; по умолчанию) или square (квадрат).
<dir> <dir>...</dir>
Создание списка каталогов, содержащего теги <li>.
Атрибуты
compact
Список делается по возможности более компактным.
<li> <li>...</li>
Начальный и конечный теги, ограничивающие элемент списка.
Атрибуты
Type = формат
Тип данного элемента списка устанавливается в соответствии с заданным форматом. Для элемента <li> в упорядоченном списке <ol>: A (прописные буквы), a (строчные буквы), I (большие римские цифры), i (маленькие римские цифры), 1 (арабские цифры; по умолчанию). Для элемента <li> в неупорядоченном списке <ul>: circle (кружок), disk (диск; по умолчанию) или square (квадрат).
Value = n
Номер данного элемента списка устанавливается равным n.
<menu> <menu>...</menu>
Задается список меню, содержащий теги <li>.
Атрибуты
compact
Список делается по возможности более компактным.
<dl> <dl>...</dl>
Создание списка определений, содержащих теги <dt> и <dd>.
Атрибуты
compact
Список делается по возможности более компактным.
<dt> <dt>...</dt>
Задается описательно - условная часть для элемента списка определений.
<dd> <dd>...</dd>
Задается описательная часть для элемента списка определений.
Организация ссылок
<a> <a>...</a>
В документе создается гиперссылка (атрибут href) или идентификатор фрагмента (атрибут name).
Атрибуты
href = url
Указывается URL целевого документа гиперссылки (необходим, если это не якорь имени).
Method = список
Задается список методов отображения, зависящих от броузера (через запятую).
Name = строка
Указывается имя идентификатора фрагмента (необходим, если это якорь гипертекстовой ссылки).
Rel = связь
Определяется связь этого документа с целевым документом.
Rev = связь
Определяется обратная связь целевого документа с данным.
Target = имя
Задается имя кадра или окна для отображения обозначенного ссылкой документа.
Title = строка
Задается заголовок целевого документа.
Urn = urn
Указывается не зависящее от места нахождения универсальное имя ресурса для данной гиперссылки.
<link> <link>
В заголовке (<head>) документа определяется ссылка из данного документа на другой документ.
Атрибуты
href = url
Указывается URL гипертекстовой ссылки целевого документа.
Method = список
Указывается (через запятую) список зависящих от броузера методов отображения для данной ссылки.
Rel = связь
Определяется связь этого документа с целевым документом. Для IE3.0 rel = style означает существование внешней таблицы стилей.
Rev = связь
Определение обратной связи целевого документа с данным.
Scr = url
Только в IE3. Указывается URL внешней таблицы стилей, которая будет использоваться для форматирования документа.
Title = строка
Задается заголовок целевого документа.
Type = text/css
Только в IE3. Задается тип внешней ссылки, которая будет использоваться как внешняя каскадная таблица стилей.
Urn = urn
Для целевого документа указывается универсальное имя ресурса, не зависящее от его места нахождения.
<base> <base>...</base>
Указывается базовый URL для всех относительных URL в данном документе.
Атрибуты
href = url

Указывается надпись для кнопки сброса.
<input> <input type=submit>
В <form> создается кнопка передачи.
Атрибуты
name=строка
Указывается имя параметра, который будет передан в приложение обработки форм при выборе данного элемента ввода. Этот атрибут является необходимым.
value=строка
Указывается надпись для кнопки передачи, а также значение параметра, посылаемого в приложение обработки форм при щелчке на этой кнопке.
<input> <input type=text>
В <form> создается элемент ввода текста.
Атрибуты
maxlength=n
Для данного элемента указывается максимальное количество вводимых символов.
name=строка
Указывается имя параметра, который передается в приложение обработки форм для данного элемента ввода. Этот атрибут является необходимым.
Size=n
Для данного элемента указывается количество отображаемых символов.
value=строка
Задается исходное значение данного элемента.
<option> <option>...</option>
Определить элемент меню внутри элемента <select> в <form>.
Атрибуты
selected
Элемент отображается как выбранный по умолчанию.
value=строка
Возвратить указанное значение данного элемента в приложение обработки форм вместо содержимого <option>.
<select> <select>...</select>
Определить в <form>меню с несколькими вариантами выбора или список прокрутки с включением одного или более тегов <option>.
Атрибуты
multiple
Пользователю разрешается выбирать в <select> более одного <option>.
name=строка
Определить текст, который будет передаваться в приложение обработки форм при выборе соответствующего значения <option>. Этот атрибут является необходимым.
Size=n
Отображение элементов меню будет организовано в виде ниспадающего меню
